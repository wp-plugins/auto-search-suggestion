This plugin will help admin of the website to filter the search of page/posts based on some criteria i.e. all the available post types.

== Description ==
1. This plugin will help admin of the website to filter the search of page/posts based on some criteria i.e. all the available post types. It will save time and avoid going through several unnecessary posts and pages, thus making the task of admin faster, smoother and smarter.
2. By using this plugin, admin can have search with suggestion of post/page's title or(all the available post types.).
3. If admin want to search with suggestion in particular post type then admin must have to check marked on particular post type.

    
== Installation ==

1. Copy the entire /auto-search-suggestion/ directory into your /wp-content/plugins/ directory.
2. Activate the plugin.
3. New Tab called Auto Search will be genereated.
4. You can setect post types.
